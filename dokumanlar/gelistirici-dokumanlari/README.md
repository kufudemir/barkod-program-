# Gelistirici Dokumanlari

Bu klasor, projeye yeni katilan bir gelistiricinin hizli onboarding yapabilmesi icin ozet dokumanlari icerir.

## Icerik
- `kurulum-rehberi.md`: adim adim kurulum ve calistirma
- `api-dokumani.md`: API endpoint gruplari, kimlik dogrulama ve ornekler
- `mimari.md`: sistem mimarisi ve katmanlar
- `ozellikler.md`: modul bazli ozellik kapsami
- `roadmap.md`: tamamlanan fazlar ve sonraki hedefler
- `ekran-goruntuleri/`: README icin proje anlatim gorselleri (PNG)

## Gorsel Uretimi
- Uretim scripti: `tools/tasarim/olustur-anlatim-gorselleri.ps1`
- Calistirma:
  - `powershell -ExecutionPolicy Bypass -File tools/tasarim/olustur-anlatim-gorselleri.ps1`
