# Geliştirme Fazları V3

## Faz 1: Kullanıcı Hesabı, Oturum ve Veri Sahipliği
Durum: `Devam Ediyor`

Tamamlananlar:
- Misafir oturumu
- Kayıtlı kullanıcı girişi / kaydı
- Şifre değiştirme
- Şifre sıfırlama
- Hesaba bağlı premium senkronu
- Firma geçmişi listeleme
- Bulut katalog geri yükleme

Kalanlar:
- Cihaz değişiminde daha güçlü veri kurtarma akışı
- Hesap düzeyinde premium görünürlüğünü genişletme

## Faz 2: Firma ve Cihaz Geçmişi Yönetimi
Durum: `Tamamlandı`

Tamamlananlar:
- Cihaz geçmişi firmaları
- Hesap geçmişi firmaları
- Firma yaşam döngüsü ayarları
- Etkin olmayan misafir firma temizliği

## Faz 3: Global Barkod Kataloğu Yönetimi
Durum: `Tamamlandı`

Tamamlananlar:
- Canonical ürün adı düzenleme
- Global ürün grubu düzenleme
- Aday isimlerden `Bu ismi kullan`
- Global ürün detayında son 20 firma fiyatı
- Kendi katalog verisini isim önerisinde önceliklendirme

## Faz 4: Admin Senkron Merkezi
Durum: `Tamamlandı`

Tamamlananlar:
- Firma / cihaz bazlı senkron özeti
- Son hata, son başarılı senkron, bekleyen durum görünümü
- Teknik batch ekranını ikinci plana alma
- Senkron merkezi admin sayfası

## Faz 5: Aydınlatma Metni ve Veri Kullanım Onayı
Durum: `Tamamlandı`

Tamamlananlar:
- Mobil uygulamada aydınlatma metni
- Web sitesinde aydınlatma sayfası
- Veri toplama ve kullanım açıklaması
- Uygulama açılışında veri kullanım onay kapısı
- Ayarlar içinde aydınlatma ve veri kullanımı metinleri
- Kayıtlı kullanıcı hesabına veri kullanım onayı senkronu

## Faz 6: Web Panelden Satış Yapabilme
Durum: `Tamamlandı`

Tamamlananlar:
- Web panelde barkod okuma
- Web sepet sistemi
- Web satış tamamlama
- Android ile aynı finans kurallarını paylaşan web POS akışı
- Web satış kayıtlarını görüntüleme
