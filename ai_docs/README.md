# AI Docs Sistemi

Bu klasör, KolayKasa projesi için AI destekli analiz, faz takibi, bağlam hafızası ve güvenli değişiklik yönetimi amaçlı oluşturuldu.

## Amaç
- Uzun süreli geliştirmede bağlam kaybını azaltmak
- Faz kurallarını görünür ve izlenebilir tutmak
- Kod değişikliklerinden bağımsız bir operasyonel hafıza katmanı sağlamak

## Bu Turda Eklenen Senkron
- `ai_docs` hariç tüm proje dökümanları tarandı
- sonuçlar `sources_catalog_*` ve `document_scan_*` dosyalarına yazıldı
- `rules.md` proje kural dokümanlarıyla güncellendi
