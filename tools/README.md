# Tools

Bu klasör proje yardımcı dosyalarını içerir.

## Klasörler
- `license-generator/`: premium lisans üretici araçları
- `web-kurulum/`: Composer, PHP ayar dosyası ve web kurulum yardımcıları
- `web-patch-yardim/`: web patch hazırlama standardı ve notları
- `arastirma/barkodbankasi-ornekleri/`: BarkodBankasi HTML örnekleri
- `tasarim/`: görsel kaynaklar

## Standart
- Kurulum araçları `web-kurulum/` altında tutulur.
- Araştırma çıktıları `arastirma/` altında tutulur.
- Görsel dosyalar `tasarim/` altında tutulur.
- Yeni yardımcı scriptler mümkünse konu bazlı alt klasör açılarak eklenir.
